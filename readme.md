## Getting Started
### Prerequisites
Firstly install npm on your device
```sh
npm install npm@latest -g
```
### Installation
1. Clone the Repo
```sh
git clone https://github.com/realmohaa/Car-Pit-Backend.git
```
2. Install NPM packages
```sh
npm install --save
```
